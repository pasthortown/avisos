15. Patron: Badge Semitransparente

Todos los badges usan el mismo patron: background: rgba(COLOR, 0.15) + color: COLOR. Esto crea un efecto de profundidad consistente sobre fondos oscuros. Para crear un badge nuevo:

/* Patron para badge custom */ .badge-custom { background: rgba(R, G, B, 0.15); /* color al 15% opacidad */ color: #RRGGBB; /* color al 100% */ } /* Ejemplo: badge naranja */ .badge-warning { background: rgba(249, 115, 22, 0.15); color: #f97316; }
